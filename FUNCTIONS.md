## `drives <drives>`
Outputs information about the specified drives/disks, if specified. Otherwise, outputs information about all drives/disks.

> [!CAUTION]
>
> Some information outputted by this command may be inaccurate in WSL!

## `volumes <drives>`
Outputs the volumes of the specified drives, as well as some information about them, if specified. Otherwise, outputs information about the volumes of all drives.

> [!CAUTION]
>
> Some information outputted by this command may be inaccurate in WSL!

## `clear`
Clear the screen.

## `exit`
Exit.
